# Documentacion de Mejoras MLB

Esta carpeta guarda un registro cronologico de cambios y mejoras aplicadas en MLB.

## Convencion de archivos
- Formato recomendado: `YYYY-MM-DD_HH-mm-ss_mejora_<tema>.md`
- Ejemplo: `2026-04-14_23-49-33_mejora_full_game.md`

## Contenido recomendado por registro
- Fecha y hora
- Cambio realizado
- Mejora observada
- Evidencia (metricas y archivo fuente)
- Impacto en produccion
- Proximo paso

## Regla de uso
Cada vez que se haga una mejora, crear un archivo nuevo con timestamp y documentar:
1. Que se cambio.
2. Que mejoro.
3. Como se valido.
